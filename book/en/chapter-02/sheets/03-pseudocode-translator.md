## WHAT YOU'LL LEARN

Pseudocode is code-like instructions in plain English. This activity teaches you to write structured thinking.

---

## PART 1: Translate Real-World to Pseudocode

**Scenario: Making lunch**

**Real thinking (messy):**
"Well, I need to check if I'm hungry. If I'm hungry, I should see what food is available. If there's good food, I'll make a meal. But if there's no good food, I might just make a snack or wait until dinner."

**Pseudocode (structured):**
```
START
  IF (I'm hungry) THEN
    Check what food is available
    IF (good food exists) THEN
      Make a meal
    ELSE
      Make a snack
    END IF
  ELSE
    Wait until dinner
  END IF
END
```

Notice: Same thinking, but organized with IF/THEN/ELSE and clear steps.

## PART 4: Find the Mistake

```
START
  IF (I'm hungry) THEN
    Get snack
    Eat snack
  END IF
END
```

**Problems with this pseudocode:**

1. What if there's NO snack available?
2. What if I'm NOT hungry?

**Rewrite it to fix these problems:**

```
START


END
```
